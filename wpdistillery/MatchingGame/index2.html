The game mechanics are simple. The user will click on an image to reveal it. It will stay revealed until the user clicks on a second image. If both images match, they will stay revealed. If they donâ€™t, they will both change back into the hidden icon. Once all of the images have been revealed the game ends.

The score is determined by the number of clicks. Each time the user clicks on one of the images, add 1 to the score (the lowest possible score is 16 if you already know where all of the images are or you are insanely lucky). When the reset button is clicked, all of the images will return to the hidden icon and the score will reset to 0
